Hvordan hoste en nettside med github pages?

Litt om github pages

Github pages er en gratis webhosting platform på github. Ved å ta ibruk det allerede eksisterende og populære repository systemer er dette den aller enkleste måten å publisere nettsiden sin. På grunn av sin simplisitet har den noen begrensinger, og at bare statiske nettsider støttes er den største.

Last opp siden til github pages

Nettsidens landingsside må hete index.html. Når du er i nettsidens “repository” på github.com trykk på “settings”, deretter “pages”. Velg “main” der som det står “none”, og til høyre skal det stå “/ (root)”. Alt du trenger for å laste opp nettsiden nå er å trykke på “Save”. Nettsiden vil straks være publisert på “https//:(github brukernavnet ditt).github.io/(navnet på repository)/”. Vær oppmerksom på at det kan ta litt tid nettsiden er lastet opp. Derfor last siden inn på nytt etter en stund dersom du ikke ser den med en gang.
